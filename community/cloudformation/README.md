# AWS CloudFormation Security Patterns

Detects security misconfigurations in AWS CloudFormation templates.

## Patterns

- `cloudformation-s3-public-access`: S3 buckets with public access
- `cloudformation-s3-no-encryption`: S3 buckets without server-side encryption
- `cloudformation-iam-lambda-assume-role`: Overly permissive IAM assume roles
