# GraphQL Security Patterns

Detects GraphQL API security misconfigurations.

## Patterns

- `graphql-introspection-enabled`: Introspection enabled in production
- `graphql-query-depth-unlimited`: Missing query depth limiting
- `graphql-field-cost-undefined`: Missing field cost analysis
- `graphql-debug-mode`: Debug mode enabled in production
