# Azure ARM Template Security Patterns

Detects security misconfigurations in Azure Resource Manager templates.

## Patterns

- `arm-azure-storage-enable-https`: Storage accounts not enforcing HTTPS
- `arm-azure-sql-no-firewall`: SQL servers without proper firewall rules
